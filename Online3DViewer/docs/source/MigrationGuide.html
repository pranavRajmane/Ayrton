Migration Guide

This document lists changes in the engine to help you to migrate your code base.

0.11.0

  • {@link CameraMode} is renamed to {@link ProjectionMode}. The corresponding parameter also changed in {@link EmbeddedViewer} from cameraMode to projectionMode.

0.8.21

  • {@link Init3DViewerElements} must be called when the document is loaded and returns an array of the created {@link EmbeddedViewer} objects. Earlier it registered a load event handler inside, now it's the responsibility of the caller.

0.8.20

  • {@link Init3DViewerElement} has been renamed to {@link Init3DViewerFromUrlList}.
  • {@link Init3DViewerFromFileList} is now available to initialize the viewer based on File objects.
